# Trip & Mind

**Author's tours in Ireland** 🌿

A modern and atmospheric website for exclusive travel experiences across Ireland.

![Trip & Mind Preview](https://github.com/user-attachments/assets/1f62396a-bd44-4e46-b89b-e0d3c583d149)

## Features

- Stunning hero section with automatic slider
- Dynamic gallery and news loaded from JSON
- Fully working contact form (submissions saved to `messages.json`)
- Completely responsive design (mobile + desktop)
- Smooth animations and elegant glassmorphism style
- Easy to run locally and share via ngrok
